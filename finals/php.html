PHP

PHP is a recursive acronym for Hypertext Preprocessor. It is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

Example:

     
    	<html>
        <head>
            <title>Example</title>
        <head>
        <body>
            <?php
                echo "Sample PHP script."
            ?>
        </body>
	</html>
