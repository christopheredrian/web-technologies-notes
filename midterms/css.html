Declarations

CSS Statements (At-Rules)

  • @charset
    • referes to the character set that the style sheet used.
  • @import
    • used when css engine includes an external style sheet.
  • @namespace
    • lets the css engine consider its contents to be prefixed with XML namespace.
  • @document
    • conditional group rule that applies contents if document that uses the style sheet meets defined conditions
  • @font face
    • describes external font aspect.
  • @keyframes
    • describes css animation sequence aspect.
  • @media (CSS Conditionals Level 3)
    • conditional group rule at which contents are applied only if device meets media query defined conditions.
  • @page
    • describes changes in layout aspect to be applied upon document printing.
  • @supports (CSS Conditionals Level 3)
    • conditional group rule that applies contents if browser meets defined conditions.

*conditional group rule - may be considered to be something like property values that can be categorized based on common syntax.

Simple Selector

  • Type Selector
    • selects elements of the document by node names.
  • Universal Selector
    • uses an asterisk (*) to refer to any type of a particular element.
  • Attribute Selector
    • selects elements based on attributes or its values.
      • [attr]
        • represents an element having attr as attribute name.
      • [attr=value]
        • represent an elements having "value" as the value of attr.
      • [attr|=value]
        • represent an elements with attr attribute having value equal to or begins with "value" followed by "-" (U+002D) for language subcode matches.
      • [attr~=value]
        • represent an element with attr attribute having set of words separated by whitespaces at which one of such words is "value" as its value.
      • [attr^=value]
        • represents an element with attribute attr having a value that begins with "value".
      • [attr$=value]
        • represents an element with attribute attr having a value that ends with "value".
      • [attr*=value]
        • represents an element with attribute attr having a value that contains "value".
      • [attr operator value i]
        • adding "I" or "i" (for characters within the ASCII range) causes case-insensitive comparison of attribute values.
  • Class Selector
    • selects elements that match based on class attribute.
  • ID Selector
    • selects elements that match based on id attribute.
  • Pseudo Selector
    • Dynamic Pseudo Class
      • link pseudo
        • :link - selects hyperlink elements that have href attribute.
        • :visited - links already visited are selected.
    • User Action Pseudo Class
      • :hover - selected as mouse hovers an element.
      • :active - selects elements usually links as they are activated.
      • :focus - select as an element is focused by keyboard.
    • Target Pseudo-class
      • :targetCSS3 - used with ID and matches if #value in the current URL matches ID.
    • Language Pseudo-class
      • :lang() - elements having lang attribute or are within some other elements that may have included the lang attribute are selected.
    • UI Elements States Pseudo-class
      • :enable - inputs in default state enabled at are selected.
      • :disabled - inputs having disabled attribute are selected.
      • :checked - selects checked checkboxes.
      • :indeterminate - selects radio buttons that are neither chosen or unchosen.
    • Structural Pseudo Class
      • :rootCSS3 - selects elements at root document.
      • :firstchild - selects first element within its parent.
      • :last-childCSS3 - selects last element within its parent.
      • :only-childCSS3 - selects only element of its parent.
      • :nth-child()CSS3 - selects nth element within its parent.
      • :first-of-typeCSS3 - selects first element of its type.
      • :last-of-type - selects last element of its type.
      • :only-of-type - selects element if it is the only element of its type.
      • :nth-of-type() - similar to nth-child but refers to nth element of a particular type.
      • :nth-last-of-type() - similar to nth-of-type() but counts from the bottom.
      • :empty - selects elements with no text.
      • :not() - selects elements that do not match what is specified in the parenthesis.
  • Combinators
    • Descendant Combinators (whitespace)
      • nodes that are children or children of children and so on of specified element are selected.
    • Child Combinators (>)
      • nodes that are children of element specified are selected.
    • Sibling Combinators
      • Adjacent Sibling Combinators (+)
        • nodes that immediately follows element specified are selected.
      • General Sibling Combinators (~)
        • nodes that follows element specified are selected.
  • Pseudo-elements
    • ::first-letterCSS3, :first-letter
      • selects first letter of text in an element.
    • ::first-lineCSS3, :first-line
      • selects first line of text in an element,
    • ::before(CSS3), :before
      • may be used to add contents before a particular element.
    • ::afterCSS3, :after
      • may be used to add contents before a particular element.

CSS Rule Precedence

  • By origin and importance
    • user agent important declarations
    • user important declarations
    • author important declarations
    • author normal declarations
    • user normal declarations
    • user agent normal declarations
  • By specificity
    • inline style
    • number of ID selectors
    • number of class selectors, attribute selectors and pseudo classes
    • number of type selectors and pseudo classes
    • number of type selectors and pseudo-elements
  • By order - when the CSS file is read from top to bottom, a style for a particular element or class overrides the style before it that applies to the same element or class.
